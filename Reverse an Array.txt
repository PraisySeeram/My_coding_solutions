Write a program to reverse an array or string

Problem
Iterative way:

Initialize start and end indexes.
start = 0, end = n-1
In a loop, swap arr[start] with arr[end] and change start and end as follows.
start = start +1; end = end â€“ 1
Input:
Each test case consists of 2 lines.
The first line takes the size of array, N.
The second line takes N space separated integers for array Note that the inputs can be either 1's or 0's.
Output:
Reverse Array
. Constraints:
1<=N<=100
Example:
Input:
6
1 2 3 4 5 6
Output:
6 5 4 3 2 1

Input:
3
58 23 44
Output:
44 23 58


Input:
4
23 57 10 99
Output:
99 10 57 23

Solution in C :-

#include<stdio.h>  
int main()  
{  
    int n, a[n],i;  
    scanf("%d", &n);  
    for(i = 0; i < n; i++)  
    {  
        scanf("%d", &a[i]);  
    }  
 
    int  Sp=0,Ep=n-1; 
    while(Sp<=Ep)
    {
       int tem = a[Sp];
       a[Sp] = a[Ep];
       a[Ep] = tem;
       Sp++;
       Ep--;
    }
    
    for(i = 0; i < n; i++)  
    {  
        printf("%d ", a[i]);  
    }  
} 

Time Complexity : O(n)
Space Complexity : O(1)
